Resource Profile: ACH Daily Event ServiceRequest

Official URL: http://www.cdc.gov/nhsn/fhirportal/dqm/ig/StructureDefinition/ach-daily-servicerequest Version: 1.0.0
Standards status: Trial-use Maturity Level: 1 Computable Name: AcuteCareHospitalDailyEventServiceRequest

This profile contains Required and Must Support data elements for reporting COVID-19, Influenza, and Respiratory Syncytial Virus (RSV) laboratory order information for the NHSN Acute Care Hospital (ACH) Daily Digital Quality Measure. This profile is based on the HL7 FHIR R4 ServiceRequest Resource v4.0.1.

Usages:

  • Examples for this Profile: ServiceRequest/servicerequest-example-ach-daily-initialpopulationpass-covid, ServiceRequest/servicerequest-example-ach-daily-negativepcr-covid-1, ServiceRequest/servicerequest-example-ach-daily-negativepcr-covid-2 and ServiceRequest/servicerequest-example-ach-daily-rsvlabbtg-rsv
