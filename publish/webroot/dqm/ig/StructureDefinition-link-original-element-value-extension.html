Extension: Original Element Value

Official URL: http://www.cdc.gov/nhsn/fhirportal/dqm/ig/StructureDefinition/link-original-element-value-extension Version: 1.0.0
Standards status: Trial-use Maturity Level: 1 Computable Name: OriginalElementValue

This extension stores the original value of an element from a source that was modified by a processing system (e.g., NHSNLink) to make it usable for NHSN. Modification may include changing logically invalid data, changing data that does not match specification requirements (this modification is used sparingly, for specific instances only), or normalizing data.

Context of Use
