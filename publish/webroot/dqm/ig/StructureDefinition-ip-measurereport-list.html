Resource Profile: NHSN Individual MeasureReport List

Official URL: http://www.cdc.gov/nhsn/fhirportal/dqm/ig/StructureDefinition/ip-measurereport-list Version: 1.0.0
Standards status: Trial-use Maturity Level: 1 Computable Name: IndividualMeasureReportList

An Individual MeasureReport List contains references to all Individual MeasureReports for a given NHSN measure; each MeasureReport pertains to an individual patient within the Initial Population of a given measure.

Usages:

  • Use this Profile: Subject-List MeasureReport Profile
  • Refer to this Profile: Subject-List MeasureReport Profile
  • Examples for this Profile: List/list-example-ach-daily-individual-measurereport-list and List/list-example-ach-monthly-individual-measurereport-list
