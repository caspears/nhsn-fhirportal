Resource Profile: NHSN Submitting Organization

Official URL: http://www.cdc.gov/nhsn/fhirportal/dqm/ig/StructureDefinition/nhsn-submitting-organization Version: 1.0.0
Standards status: Trial-use Maturity Level: 1 Computable Name: SubmittingOrganization

This profile includes the name and identifier of the organization submitting the report Bundle, and may include other information about the organization.

Usages:

  • Use this Profile: NHSN MeasureReport Bundle and NHSN Submission Header Bundle
  • Examples for this Profile: Example NHSN Submitter
