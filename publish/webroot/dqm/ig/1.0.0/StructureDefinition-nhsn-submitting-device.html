Resource Profile: NHSN Submitting Device

Official URL: http://www.cdc.gov/nhsn/fhirportal/dqm/ig/StructureDefinition/nhsn-submitting-device Version: 1.0.0
Standards status: Trial-use Maturity Level: 1 Computable Name: NHSNSubmittingDevice

This profile includes the name and version of the device or software submitting the report Bundle, and may include other information about the device.

Usages:

  • Use this Profile: NHSN MeasureReport Bundle and NHSN Submission Header Bundle
  • Examples for this Profile: Device/device-example-submitting-device

You can also check for usages in the FHIR IG Statistics
